# 🚀 TrollTouch 快速开始指南

## 📦 安装

1. **下载 IPA**
   - 从 GitHub Actions 下载编译好的 `TrollTouch.ipa`

2. **通过 TrollStore 安装**
   - 打开 TrollStore
   - 点击 "+" 按钮
   - 选择下载的 IPA 文件
   - 等待安装完成

## 🎮 使用方法

### 方式 1：定时自动运行（推荐）

1. **打开 TrollTouch 应用**
2. **设置工作时间**
   - 开始时间：例如 `9` (9:00)
   - 结束时间：例如 `18` (18:00)
3. **启用定时任务**
   - 打开「启用定时任务」开关
4. **保存设置**
   - 点击「保存设置」按钮
5. **完成！**
   - 应用会在设定时间自动启动
   - 超出时间自动停止

### 方式 2：手动立即运行

1. **打开 TrollTouch 应用**
2. **点击「🚀 立即启动」**
3. **确认启动**
4. **开始运行！**

### 停止运行

- 点击「⏹ 停止」按钮
- 或关闭定时任务开关

## 🔍 查看日志

### 方法 1：系统日志（推荐）
```bash
# Windows PowerShell
idevicesyslog | Select-String "XCTestRunner"

# Mac/Linux
idevicesyslog | grep XCTestRunner
```

### 方法 2：应用内日志
应用会在控制台输出详细日志，包括：
- 测试启动状态
- 每个操作的执行情况
- 错误信息

## ⚙️ 配置说明

### 工作时间
- **格式**：24小时制（0-23）
- **示例**：
  - 白天工作：9:00 - 18:00
  - 夜间工作：22:00 - 6:00（跨午夜）

### 自动化行为
当前版本的自动化行为：
- 观看视频：3-8秒随机
- 点赞概率：30%
- 关注概率：5%
- 滑动方向：从下往上

## 🐛 故障排查

### 问题：点击启动后没有反应
**解决方案：**
1. 检查 TikTok 是否已安装
2. 查看系统日志确认错误
3. 尝试手动打开 TikTok 测试

### 问题：定时任务没有自动启动
**解决方案：**
1. 确认「启用定时任务」开关已打开
2. 检查当前时间是否在工作时间内
3. 查看系统通知是否有提示

### 问题：自动化运行一段时间后停止
**解决方案：**
1. 检查是否超出工作时间
2. 查看日志确认是否有错误
3. 尝试重新启动应用

## 📱 系统要求

- **iOS 版本**：14.0 或更高
- **TrollStore**：已安装并激活
- **TikTok**：已安装（国际版或国内版）
